# 実装計画 - タイトルの下に説明文を表示する

この計画では、ブログ記事のタイトルの下に説明文を表示するための変更の概要を説明します。

## 提案される変更

### [ブログ投稿テンプレート]
#### [変更] [slug.astro](file:///Users/ofuchirin/code/lantern-tech-blog/astroprojects/src/pages/[lang]/blog/[slug].astro)
- `<header>` セクションを更新し、`entry.data.description` が存在する場合に `<p>` タグ内にレンダリングするようにします。
- 新しい `.description` クラスのCSSスタイルを追加します。
- 説明文は `<h1>` タイトルの直後に表示されます。

## 検証計画

### 手動検証
1.  開発サーバーを起動します (`npm run dev`)。
2.  フロントマターに `description` があるブログ記事（例: `/jp/blog/integrate`）に移動します。
3.  説明文がタイトルの下に表示されていることを確認します。
4.  説明文がないブログ記事（ある場合）に移動し、空の要素がレンダリングされていないことを確認します。
5.  説明文のスタイルを確認します。
