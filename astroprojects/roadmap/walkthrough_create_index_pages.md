# インデックスページ作成のウォークスルー

ブログ、発表資料、および参考文献カテゴリのインデックスページを作成しました。また、ユーザーのリクエストによりページタイトルを左揃えに変更しました。

## 変更点

### 1. 新しいコンポーネント: `ArticleCard.astro`
以下を表示する再利用可能なコンポーネントを作成しました：
- **サムネイル**: フロントマターで定義されたOGP画像を使用します。
- **タイトルと説明**: 記事へのクリック可能なリンクです。
- **著者**: 著者名を表示します。

### 2. 新しいインデックスページ
各カテゴリのインデックスページを作成し、記事をレスポンシブなグリッドレイアウトで一覧表示するようにしました。
また、ページタイトル「ブログ」「発表資料」「参考文献」を中央揃えから**左揃え**に変更しました。

- `src/pages/[lang]/blog/index.astro`
- `src/pages/[lang]/newsrelease/index.astro`
- `src/pages/[lang]/notes/index.astro`

## 検証結果

### ブラウザ検証
ブラウザサブエージェントを使用してページを検証しました：
- **ブログインデックス (`/jp/blog`)**: ブログ記事が正しく表示され、タイトルが左揃えになっています。
- **発表資料インデックス (`/jp/newsrelease`)**: 発表資料が正しく表示され、タイトルが左揃えになっています。
- **参考文献インデックス (`/jp/notes`)**: 参考文献が正しく表示され、タイトルが左揃えになっています。
